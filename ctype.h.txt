islower
检查字符 c 是否是小写字母（a-z）
如果 c 是小写字母，返回非零值（通常是1）,如果 c 不是小写字母，返回0
(tolower <string.h>中大写字符转为小写字符的标准库函数）

isupper
检查字符 c 是否是大写字母（A-Z）
如果 c 是大写字母，返回非零值（通常是1）,如果 c 不是大写字母，返回0
(toupper <string.h>中小写字符转为大写字符的标准库函数）

isalpha
判断是否是字母a~z或A~Z

isdigit
判断是否是十进制数字0~9

isxdigit
判断是否是十六进制数字
包含所有的十进制数字，小写字母a~f，大写字母A~F

isalnum
判断是否是字母或数字 0~9,a~z,A~Z
